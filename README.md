# 연령별 인구 구조 시각화 (Population Age Structure Viewer)

행정안전부 "연령별 인구현황" 월간 데이터를 이용해, 특정 지역(읍·면·동 단위 포함)의
연령별 인구 구조를 확인하고, 전국에서 인구 구조(연령 분포 모양)가 가장 비슷한
지역 Top 5를 찾아주는 Streamlit 웹앱입니다.

## 데모 화면 구성

1. 사이드바에서 지역명을 검색(부분 일치) → 드롭다운에서 선택
2. 선택한 지역의 총인구수 / 남·여 비율 요약 지표
3. 선택 지역의 연령별 인구 구조(계/남/여) 꺾은선 그래프 (Plotly)
4. 전국에서 인구 구조가 가장 비슷한 지역 Top 5 표 + 비교 꺾은선 그래프

## 데이터

- 파일명: `202606_202606_연령별인구현황_월간.csv`
- 원본 인코딩: `CP949`
- 원본 파일의 마지막 행이 손상(중간에 끊김)되어 있어, 해당 행은 로드 시 자동으로 건너뜁니다.
- 숫자 컬럼은 천 단위 구분 콤마(`,`)가 포함된 문자열이라 콤마 제거 후 숫자형으로 변환합니다.
- 컬럼 구조 예시: `2026년06월_계_0세`, `2026년06월_남_100세 이상`, `2026년06월_여_총인구수` 등
  (연월 부분은 파일 갱신 시에도 정규식으로 자동 인식됩니다.)

## 주요 기능

### 1. 지역 선택 (검색 + 선택)
텍스트 입력으로 지역명을 검색해 후보를 좁히고, 드롭다운에서 최종 선택합니다.
읍·면·동부터 시·군·구, 시·도 단위까지 전체 행정구역이 검색 대상입니다.

### 2. 연령별 인구 구조 그래프
선택한 지역의 0세 ~ 100세 이상 연령별 인구수를 계/남/여 3개 라인으로
Plotly 꺾은선 그래프에 표시합니다.

### 3. 인구 구조 유사 지역 Top 5
- 각 지역의 연령별 인구를 총인구 대비 **비율**로 정규화한 뒤,
  선택 지역과의 **코사인 유사도(cosine similarity)**를 계산합니다.
- 총인구 규모가 크게 달라도 연령 분포의 "모양"이 비슷하면 높은 유사도로 나타납니다.
- 유사도 상위 5개 지역을 표로 보여주고, 선택 지역(굵은 검은선)과 함께
  연령별 인구 비율(%) 그래프로 겹쳐서 비교합니다.
- 총인구가 0인 지역은 비교 대상에서 제외됩니다.

## 파일 구조

```
.
├── app.py                                   # Streamlit 앱 코드
├── requirements.txt                         # 의존성 목록
└── 202606_202606_연령별인구현황_월간.csv      # 데이터 파일 (같은 폴더에 위치)
```

## 로컬 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Community Cloud 배포 방법

1. `app.py`, `requirements.txt`, 데이터 CSV 파일 3개를 같은 폴더 구조로 GitHub 저장소에 올립니다.
2. [share.streamlit.io](https://share.streamlit.io) 에서 저장소를 연결합니다.
3. Main file path를 `app.py`로 지정하고 배포합니다.

## 사용 라이브러리

- [Streamlit](https://streamlit.io/) — 웹앱 프레임워크
- [Pandas](https://pandas.pydata.org/) — 데이터 처리
- [NumPy](https://numpy.org/) — 유사도(코사인 유사도) 계산
- [Plotly](https://plotly.com/python/) — 인터랙티브 그래프
